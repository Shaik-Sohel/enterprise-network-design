# Enterprise Network Design

## Project Overview
This project demonstrates the design and implementation of an enterprise-level network infrastructure using Cisco Packet Tracer. The network was configured with routers, switches, VLANs, static routing, and dynamic routing protocols to ensure efficient communication between multiple departments.

## Technologies Used
- Cisco Packet Tracer
- Routing and Switching
- VLAN Configuration
- Static Routing
- Dynamic Routing (RIP/OSPF)
- IPv4 Addressing
- DHCP

## Features Implemented
- Designed enterprise network topology using routers and switches
- Configured VLANs for departmental network segmentation
- Implemented trunking between switches
- Configured inter-VLAN routing for communication between VLANs
- Implemented static routing between networks
- Configured dynamic routing protocols such as RIP and OSPF
- Configured DHCP for automatic IP address assignment
- Performed network troubleshooting and connectivity testing

## Network Devices Used
- Cisco Routers
- Cisco Switches
- PCs and End Devices

## VLAN Configuration
- Created VLANs for different departments
- Assigned switch ports to VLANs
- Configured trunk ports between switches
- Verified VLAN communication

## Routing Configuration
### Static Routing
- Configured static routes for manual path selection
- Verified connectivity using ping and traceroute

### Dynamic Routing
- Configured RIP and OSPF protocols
- Advertised networks dynamically
- Verified routing table updates

## Testing and Verification
- Performed ping tests between networks
- Verified inter-VLAN communication
- Checked routing tables using CLI commands
- Troubleshooted connectivity issues
