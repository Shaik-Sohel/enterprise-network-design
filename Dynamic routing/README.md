# Dynamic Routing Configuration Lab

## Project Overview
This project demonstrates dynamic routing configuration using Cisco Packet Tracer. The network was designed using routers, switches, and PCs to enable communication between multiple networks automatically using routing protocols.

## Technologies Used
- Cisco Packet Tracer
- Dynamic Routing
- RIP Protocol
- OSPF Protocol
- Routers and Switches
- IPv4 Addressing

## Features Implemented
- Configured routers for dynamic routing
- Implemented RIP routing protocol
- Configured OSPF routing protocol
- Established communication between different networks
- Verified routing tables and connectivity
- Performed troubleshooting using ping and traceroute

## Network Devices Used
- Cisco Routers
- Cisco Switches
- PCs

## Routing Protocols
### RIP (Routing Information Protocol)
- Configured RIP Version 2
- Advertised connected networks
- Verified route propagation

### OSPF (Open Shortest Path First)
- Configured OSPF process IDs
- Assigned networks to OSPF areas
- Verified neighbor relationships

## Testing and Verification
- Ping testing between PCs
- Traceroute verification
- Checked routing tables using CLI commands

- OSPF and RIP Setup
- Router Configuration
- Network Troubleshooting
- Routing Table Analysis
